Etiquetas de Meta Datos

Las Etiquetas de Meta Datos son elementos que se colocan dentro de la sección del documento HTML. Su función es proporcionar información ("metadatos") sobre el HTML mismo, como quién lo creó, qué contiene, cómo debe mostrarse o qué recursos externos necesita. Esta información no es visible en el cuerpo de la página, pero es crucial para los navegadores, motores de búsqueda (SEO) y redes sociales.

Etiquetas de Agrupación

Las Etiquetas de Agrupación (o elementos de bloque) son aquellas que se utilizan para estructurar y organizar grandes secciones de contenido dentro del de tu documento HTML. Por defecto, estas etiquetas ocupan todo el ancho disponible y comienzan en una nueva línea, separando su contenido del resto. Define una división o sección genérica. Es el contenedor de bloque más común para agrupar elementos y aplicar estilos con CSS cuando no hay una etiqueta semántica más apropiada

Etiquetas Textuales

Las Etiquetas Textuales (o elementos en línea) se utilizan para dar formato, significado o funcionalidad a una pequeña porción de texto dentro de un bloque más grande, como un párrafo. A diferencia de las etiquetas de agrupación, estas etiquetas no comienzan una nueva línea ni ocupan todo el ancho; solo envuelven el contenido que necesitan modificar. Define una sección genérica en línea. Se usa para aplicar un estilo específico con CSS o JavaScript a una pequeña parte de texto cuando no hay una etiqueta semántica más adecuada.
